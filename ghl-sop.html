Before you start: Have your PropStream export ready, skip-traced, and Truelist-cleaned. Never upload a list that hasn't been cleaned. Note: risky/accept_all from Yahoo/AOL is normal for this demographic — keep those, only remove invalid/bounce statuses.
Phase 1 — List Prep
Pull list from PropStream manual
Filter: absentee owners + long holds (5+ years) + delinquent taxes. Property type = land. Export CSV with all contact fields.
Skip trace the list manual
Upload to BatchSkipTracing or similar. Get phone + email for every contact. Download results CSV.
Clean list via Truelist verify
Upload to truelist.io → run email verification → download valid-only results. Keep risky/accept_all — normal for Yahoo/AOL. Remove invalid + bounce statuses only.
Run domain health check verify
truelist.io/tools → Domain Reputation Checker → check mail.solterrarei.com. Fix any issues before sending.
Phase 2 — Format & Import
Format CSV for GHL manual
Required columns: First Name, Last Name, Email, Phone, Address, City, State, Zip, Tags.
Import to GHL manual
Contacts → Import → "Create and update contacts" → map fields → review → import. Wait for completion.
Tag by market and source manual
Apply tags: market name, source (PropStream), date (Mar-2026), Seller-Lead. Use bulk tag tool.
Verify import count verify
Check GHL contact count before and after. Confirm match minus duplicates.
Phase 3 — Campaign Setup
Assign to correct email campaign manual
GHL → Marketing → Campaigns → select market campaign → Add Contacts → filter by market tag → add all.
Check send settings verify
Domain = mail.solterrarei.com · From = Sean · Times = 9am–5pm EST weekdays · Unsubscribe present · Daily cap 200–300/day for new markets.
Verify Workflow 1 is published auto
"Campaign Reply – Lead Sorting" must be PUBLISHED. Stops sequences on reply, creates task for Sean.
Verify Workflow 2 is published auto
"New Lead – Form Submit → Pipeline + SMS to Sean" must be PUBLISHED.
Phase 4 — Launch & Monitor
Send test email to yourself verify
Send one test to sean@solterraacquisitions.com. Check: renders correctly, unsubscribe works, from name/domain correct.
Launch + monitor day 1 verify
Start campaign. Check after 2 hours: open rate ≥ 20%, bounce rate < 2%. If bounces spike → pause and re-clean immediately.
